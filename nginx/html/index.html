Gói “Khóa SSH”

Luật bảo vệ

  • Web / DOS: Nếu 1 IP gửi hơn 50 request vào web trong 5 giây, IP đó bị chặn trong 5 phút.
  • SSH: Nếu 1 IP nhập sai mật khẩu SSH 5 lần, IP đó bị chặn trong 5 phút.
  • Test nhanh: HTTP: curl http://localhost:8080/
    SSH: ssh root@localhost -p 2222
